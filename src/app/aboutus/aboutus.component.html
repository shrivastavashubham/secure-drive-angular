The Secure Drive is a part of MSC computer Science project on minimizing Insider datatheft in cloud storage through disinformation attacks. This Project work aims at Improving security from insider datatheft attacks in cloud storage through a diffrent approach. Identifying a possible masquerade attacker,Verifying his identity and launching a disinformation attack are three main objectives of this project
